# Z06-ER2-NWI

##### Add LED Dot-Matrix 400 Font
```css
@font-face {
    font-family: 'LED Dot Matrix';
    src: url('https://thedaos.github.io/Z06-ER2-NWI/LED%20Dot-Matrix%20400.ttf') format('truetype');
}
body {
    font-family: 'LED Dot Matrix', monospace;
}
```

@font-face {
